Why I should get a Desktop

By: Gregory Podolyan

Hello Mama and Papa, This is my first website coded with HTML, CSS, and JS (Java Script). I recently found a liking to these programing languages instead of using Python. But anyways, I am using this website to convince you to get the Desktop, and even better, sooner rather than later. With hardware like of a desktop’s, I could create more stuff like this and have more access to libraries with programing. But, that isn’t it, I could run Microsoft Flight Simulator, force me to be at my desk, help me make more stuff like this, and it won’t shut down whenever it feels like it.

Just to establish this, I don’t need it now. When Papa finally moves out of my room, and when mama starts to add more stuff to the house, then it would be a good time, plus you could consider this a very very very early birthday present. Now that we have that out of the way, I would like to talk about the immediate benefits, (Yes, there are long term benefits) First, I can run my favorite games/activities such as Microsoft Flight Simulator, and Vscode with Github open (My current computer can barley handle that), Secondly, I would be forced to be seated at a desk, so my posture can improve. And finally, A P.C can do almost anything from hosting a server/website by itself, to having a virtual machine that can practically run any Operating system from Windows, all the way to Kali Linux, or Unbuntu. And for the long term benefits, it’s just cheaper, instead of buying a new macbook every 3 - 5 years, I can just buy some new parts and re-install them. Powerfull, Simple, and Cost-Effective, what more could you ask for?

But what about screen time? Firstly, no staying on screens past 10PM, and considering the weather isn’t pouring down on me, I can go to Osage, hang out with friends, host soccer/social events, and just fly the RC plane around. But what about weekends? Well take last weekend into account, I barely touched my computer, because I was too busy at the park, with my friend Matthew. And finally, I can go to the gym, and start either swimming or basketball. So what do you say?
